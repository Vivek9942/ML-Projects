 Project 1: Insurance Cost Prediction
 Description :-

This project focuses on predicting medical insurance charges based on factors like age, BMI, number of children, smoking habits, and region.

🛠️ Technologies Used
Python
NumPy
Pandas
Matplotlib / Seaborn
Scikit-learn

📊 Workflow
Data collection and loading
Data preprocessing (handling categorical values, encoding)
Exploratory Data Analysis (EDA)
Model training using Linear Regression
Model evaluation using metrics like R² score

📈 What I Learned
Regression techniques
Feature importance
Data visualization
Handling real-world datasets

Project 2: Heart Disease Prediction
Description :-

This project predicts whether a person is likely to have heart disease based on medical attributes such as age, sex, chest pain type, blood pressure, cholesterol level, etc.

🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn

📊 Workflow
Data preprocessing and cleaning
Feature selection
Model training using classification algorithms (Logistic Regression / Decision Tree)
Model evaluation using accuracy score

📈 What I Learned
Classification algorithms
Model accuracy and evaluation
Importance of healthcare data analysis
Feature correlation
